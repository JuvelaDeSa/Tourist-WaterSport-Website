**Overview** 
This is a static website showcasing different water sport activities available at a popular tourist destination in Goa, India. The site includes multiple pages with information, images, and navigation for tourists interested in adventure activities.

**Features**
- Display of multiple water sports activities with descriptions and images.
- Responsive design that works on desktop and mobile devices.
- Navigation across multiple pages for easy access to activity information.
- Image gallery for better visual representation of activities.
